1.Отобразить подключённый девайс в консоли:
./adb devices


2.Вывести адрес приложения todolist в системе Android:
./adb shell pm list packages|grep todolist


3.Установить .apk файл приложениия todolist на телефон с компьютера через  ADB:
./adb install C:android/001_ToDoList-master/app/build/outputs/apk/debug/app-debug.apk


4.Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде:
./adb shell screencap storage/emylated/0/DCIM/todolist_screen1.png&&./adb pull storage/emylated/0/DCIM/todolist_screen1.png D:001_ToDoList-master


5. Вывести в консоль логи приложения todolist:
./adb logcat -c(чистим логи)
./adb logcat|grep -r "com.android.todolist"


6. Скопировать логи приложения todolist на компьютер:
./adb logcat|grep -r "com.android.todolist"D:001_ToDoList-master


7. Удалить приложение todolist с телефона через ADB:
./adb uninstall com.android.todolist
